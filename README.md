## navbar-1212
Demo App for 1212.mn

## Used technologies
Spring Boot 2.3.0
MSSQL 2019
Angular 9.1.9
Docker 3.2

## Requirements
Docker >= 3
PowerShell

## Download
```bash
git clone https://github.com/tsendee0409/navbar-1212.git
cd navbar-1212
```

## Run
```bash
powershell -file init.ps1
```

## Launch
`http://localhost:38080/`
